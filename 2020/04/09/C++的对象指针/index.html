C++中的的对象指针

指针...懂得都懂

一、指向对象的指针

# include 
using namespace std;

class A {
public:
    A(){}
};

int main() {
    // name 即为一个对象指针，此时指向一个类型为A的对象
    A * name = new A;

    return 0;
}

二、指向对象成员的指针

对象由成员构成，一整个对象的总内存为各个数据成员占据的内存区的总和。

那么对象的成员也有地址，指向对象成员地址的指针变量即为指向对象成员的指针

1、指向对象公有数据成员

# include 
using namespace std;

class A {
public:
    int num;

    A(){
        num = 1;
    }

};

int main() {
    A aa;

    int * p = &aa.num;        // 指针变量p指向aa这个对象的成员数据num

    return 0;
}

2、指向对象成员函数

先复习一下普通情况下一个指向函数的指针是什么样子的

数据类型名 （*指针变量名)(参数列表）；

例如： void (*p)( ); //p指向void型函数的指针变量；

​ p = fun; //fun入口地址赋予p；

​ (*p)( ); //调用fun函数；

# include 
# include 
using namespace std;

void make(string str) {
    cout << str << endl;
}

int main() {
    void (*p) (string) = make;

    (*p)("你好");

    return 0;
}
/*
    输出结果：
    > 你好
*/

——正题来了——

指向对象成员函数的指针的步骤

①定义指向成员函数的指针变量

数据类型 (类名::*指针变量名)(形参表);

  • 数据类型 —— 成员函数的类型
  • 类名 —— 这个对象所属的类
  • 形参表 —— 与指向成员函数的形参表一致

②取成员函数的地址

&类名::成员函数名

实例↓

# include 
# include 
using namespace std;

class A {
public:
    int num;

    A(){
        num = 1;
    }

    void eat(string a) {
        cout << "eat something" << endl;
    }

    void eat() {
        cout << "eat nothing" << endl;
    }
};

int main() {
    A aa;

    // 1、定义指针
    void (A::*p)( );        
    // 2、指向成员函数
    p = &A::eat;
    // 3、调用这个指针指向的成员函数
    (aa.*p)( );

    // 1、定义指针
    void (A::*q)(string);
    // 2、指向成员函数
    q = &A::eat;
    // 3、调用这个指针指向的成员函数
    (aa.*q)("hello world");

    return 0;
}
/*
    输出结果：
    > eat nothing
    > eat something
*/

3、this指针

说明：

C++是通过this指针实现不同对象使用同一个函数代码段分别对不同对象数据进行操作的，当调用对象时，this指针指向该对象，成员函数便访问其成员。

对象所占的空间大小只取决于该对象中数据成员所占的空间，而与成员函数无关。不论成员函数是否在类内定义，亦或是否声明为inline，成员函数均不占用对象的存储空间。

那么当不同对象的成员函数访问数据成员时，怎么保证访问的就是当前这个对象的数据成员呢？

其实每个成员函数中都包含一个特殊的指针this，这个指针指向本对象

C++ 通过编译程序，在对象调用成员函数时，把对象的地址赋予this指针，用this指针指向对象，实现了用同一个成员函数访问不同对象的数据成员。
